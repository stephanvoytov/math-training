# Математика ОГЭ — тренажер

Интерактивный тренажер для подготовки к ОГЭ по математике. Построен на React + TypeScript с адаптивным дизайном и мгновенной обратной связью.

## Возможности

- **Разбор тем** — все основные разделы математики ОГЭ: алгебра, геометрия, вероятность
- **Тестирование** — случайные задания с автоматической проверкой
- **Прогресс** — отслеживание результатов по каждой теме
- **Адаптивная верстка** — работает на десктопе, планшете и телефоне

## Технологии

- **Frontend:** React 19, TypeScript, React Router 7
- **Сборка:** Vite 8
- **Деплой:** GitHub Pages

## Запуск

npm install
npm run dev

## Сборка и деплой

npm run build
npm run deploy
